Sample: KPI Calculation (Code)

from decimal import Decimal

def net_interest_margin(interest_income, interest_expense, earning_assets):
    """NIM = (Interest Income - Interest Expense) / Average Earning Assets"""
    ii = Decimal(str(interest_income))
    ie = Decimal(str(interest_expense))
    ea = Decimal(str(earning_assets))
    return (ii - ie) / ea

nim = net_interest_margin(1_240_000, 430_000, 21_000_000)
print(f"NIM: {nim:.4%}")

Financial Math (KaTeX)

Net Present Value:

$$\text{NPV} = \sum_{t=0}^{T} \frac{CF_t}{(1+r)^t}$$

Black–Scholes (call):

$$C = S_0\Phi(d_1) - Ke^{-rT}\Phi(d_2)$$

with \(d_{1,2} = \frac{\ln(S_0/K) + (r \pm \tfrac{1}{2}\sigma^2)T}{\sigma\sqrt{T}}\)
